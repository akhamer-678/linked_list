## CS505 Linked List Project

This is a simple C++ project implementing a singly linked list with common operations.

## Features

- Insert a node (first, after, before, end)
- Delete a node (first, end, specific)
- Update node data or key
- Traverse and display the list
- Search by key or data
- Ordered insertion
- Clear the list (`makelistempty`)

## Files

- `list.h`  Header file with the `list` class declaration  
- `list.cpp`  Implementation of the linked list class  
- `app.cpp`  Example main program to test the list operations
